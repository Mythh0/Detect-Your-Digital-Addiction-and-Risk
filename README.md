# Detect Your Digital Addiction and Risk

**Detect Your Digital Addiction and Risk** is an AI-powered application that analyzes user behavior patterns to identify signs of digital addiction and excessive screen usage. The system combines facial emotion analysis, device activity monitoring, and machine learning techniques to evaluate digital well-being and provide personalized risk assessments.

## Features

* 📱 Real-time digital activity monitoring
* 😊 Facial emotion detection and tracking
* 🧠 AI-based addiction risk prediction
* 💻 Cross-device communication between laptop and mobile
* 📊 User behavior analytics and insights
* ⚡ Real-time data processing using Flask APIs
* 🔒 Privacy-focused local data analysis

## Technologies Used

* Python
* Flask
* Flask-CORS
* Scikit-learn
* NumPy
* Requests
* Psutil
* MediaPipe Face Landmark Detection

## Project Structure

* `app.py` – Main Flask application
* `emotion_tracker.py` – Emotion detection and analysis
* `model.py` – Machine learning model logic
* `laptop_bridge.py` – Laptop-side data communication
* `phone_bridge.py` – Mobile device communication
* `face_landmarker.task` – Face landmark detection model

## How It Works

1. Collects device usage and behavioral data.
2. Tracks facial expressions and emotional patterns.
3. Processes the collected information using machine learning algorithms.
4. Calculates addiction risk levels.
5. Generates insights and recommendations for healthier digital habits.

## Future Enhancements

### Objective

The goal of this project is to help users understand their digital habits, detect early signs of digital addiction, and encourage healthier technology usage through AI-driven insights.
